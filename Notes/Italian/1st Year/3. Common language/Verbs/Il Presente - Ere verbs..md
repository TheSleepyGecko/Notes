# The Present Tense

### Scriv~~ere~~

> [!Help] 
> This is the same as <mark class="hltr-red">Are</mark> verbs. Once you find a word ending in <mark class="hltr-orange">Ere</mark>, and you wish to change it to present tense replace the <mark class="hltr-orange">Ere</mark> at the end of the word with the corresponding ending in the <mark class="hltr-orange">Ere</mark> column.


|         | Ere   | Scrivere  | English            |
| ------- | ----- | --------- | ------------------ |
| Io      | -O    | Scrivo    | I write            |
| Tu      | -I    | Scrivi    | You Write          |
| Lui/Lei | -E    | Scrive    | He/She Writes      |
| Noi     | -IAMO | Scriviamo | We write           | 
| Voi     | -ETE  | Scrivete  | You write (plural) |
| Loro    | -ONO  | Scrivono  | They write         |
