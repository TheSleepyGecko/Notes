**Masculine:**

| Word | When to use                                             |
| ---- | ------------------------------------------------------- |
| Un   | when the word begins with a vowel of constanant                         |
| Uno  | when the word begins with "s + constanant" or "z" | 

**Feminine:**

| Word | When to use                 |
| ---- | --------------------------- |
| Un'  | When it starts with a vowel |
| Una  | when it starts with a constanant                            |

**Examples:**

- Una penna
- Un Diario
- Una Scuola
- Un Telefono
- Una Lavagna
- Un Quaderno
- Un Astuccio
- Un Libro
- Una Ragazza
- Una Scuola
- Uno studente
- Un' Anatra
- Un gelato
- Una professoressa
- Una bottiglia
